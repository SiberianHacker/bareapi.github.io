Пример отправки изображения на Bare API с использованием Python.

									import requests
from base64 import b64encode
import time

API_KEY = "Ключ Bare API"
SITE = "http://5.42.211.111/"
GET_DELAY = 0.8

def image_to_base64(image_path):
    with open(image_path, "rb") as image_file:
        return b64encode(image_file.read()).decode('utf-8')

def solve_captcha(filename):
    try:
        base64_image = image_to_base64(filename)

        post_response = requests.post(
            f"{SITE}in.php",
            data={
                "key": API_KEY,
                "method": "base64",
                "body": base64_image
            },
            timeout=10
        )

        if "OK|" not in post_response.text:
            print("Ошибка отправки капчи:", post_response.text)
            return None

        captcha_id = post_response.text.split("|")[1]
        time.sleep(GET_DELAY)

        get_response = requests.get(
            f"{SITE}res.php",
            params={
                "key": API_KEY,
                "action": "get",
                "id": captcha_id
            },
            timeout=10
        )

        if "OK|" not in get_response.text:
            print("Капча ещё не решена:", get_response.text)
            return None

        return get_response.text.split("|")[1]

    except Exception as e:
        print("Ошибка:", str(e))
        return None

if __name__ == "__main__":
    captcha = input("Введите имя файла: ")
    solved_captcha = solve_captcha(captcha)
    print(f"Решена капча: {solved_captcha}")

Разбор кода

Импорты

										import requests
from base64 import b64encode
import time
  • requests — HTTP-запросы к Bare API
  • base64 — преобразование изображения в base64
  • time — задержка перед получением результата

Конфигурация

										API_KEY = "Ключ Bare API"
SITE = "http://5.42.211.111/"
GET_DELAY = 0.8
  • API_KEY — ваш ключ доступа
  • SITE — адрес Bare API
  • GET_DELAY — время ожидания перед запросом результата

Функция image_to_base64

Читает файл изображения и кодирует его в Base64, необходимый для передачи через API. 

										def image_to_base64(image_path):
    with open(image_path, "rb") as image_file:
        return b64encode(image_file.read()).decode('utf-8')

Функция solve_captcha

Основная функция решения капчи:

  1. Преобразует изображение в base64
  2. Отправляет капчу через in.php
  3. Получает ID капчи
  4. Запрашивает результат через res.php

Отправка изображения

										POST /in.php
key=API_KEY
method=base64
body=BASE64_IMAGE

Получение результата

										GET /res.php
key=API_KEY
action=get
id=CAPTCHA_ID

Примечания

  • При сложных капчах можно увеличить GET_DELAY
  • Пока поддерживаются только PNG формат

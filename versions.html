Bare API версии ключей

API V1 - Общая модель для капч, где написан англ. текст

API V1.1 - Версия, использующая ChatGPT для распознавания

API V2 - Капчи русским текстом (Например: двести сорок три)

API V3 - Капчи с русскими буквами (Например: чхЩ, рРп)

API V4 - Капчи-Пазлы, для того чтобы бот крутил рамки

API V5 - Капчи, где нужно крутить 3x3 рамки чтобы увидеть текст

API V52 - Капчи с русским текстом формата 3x3, где нужно вращение

API V53 - Капчи 2x2, где нужно вращать рамки, чтобы увидеть текст

API V6 - Модель капч, обученная на капчах с сайтов

API B1 - Версия которая решает капчи квадратным шрифтом

Для использования нужной версии API кроме V1

Просто припишите её в начало используемого в коде ключа

Например:

V1.1_bjdvT3HmquOPTUEEjMbwUZQYt

V52_bjdvT3HmquOPTUEEjMbwUZQYt

B1_bjdvT3HmquOPTUEEjMbwUZQYt

Для использования V1 - приписывать версию не нужно
